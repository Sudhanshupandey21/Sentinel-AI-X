# 🚀 Sentinel AI-X

> Autonomous Decision Intelligence + Predictive Threat System

Sentinel AI-X is an advanced AI-powered defense system that not only detects threats but also predicts, simulates, and assists in decision-making.

---

## 🔥 Key Features

- 🔮 Predictive Threat Analysis  
- 📍 Real-time Movement Tracking  
- 🚨 Smart Alert System (Sound + UI Feedback)  
- 🚓 Nearby Unit Response Simulation  
- 🧠 AI-Based Recommendations  
- 🗺️ Interactive Map Visualization  

---

## 🛠️ Tech Stack

- React + TypeScript  
- Tailwind CSS  
- Leaflet (Map Integration)  
- AI Logic (Simulation-based)  

---

## 🧠 How It Works

1. Input (Image / Data)
2. AI Analysis
3. Risk Prediction
4. Simulation
5. Alert + Response System
